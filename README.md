# 🍕 Pizza Sales Analysis Dashboard

## 📊 Project Overview

This project analyzes pizza sales data using Python and Power BI to extract business insights.

## 🚀 Tools Used

* Python (Pandas, Matplotlib)
* Power BI
* SQL (optional)

## 📁 Dataset

* 48,000+ sales records
* Includes order details, pricing, categories, and timestamps

## 📊 Key Insights

* Identified top-selling pizzas
* Analyzed peak ordering hours
* Compared revenue by category and size
* Calculated average order value

## 📈 Dashboard Features

* KPI metrics (Revenue, Orders, Quantity, AOV)
* Sales trends over time
* Peak hour analysis
* Category & size breakdown
